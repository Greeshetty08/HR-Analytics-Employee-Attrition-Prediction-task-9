# HR Analytics - Employee Attrition Prediction

## Project Overview

This project focuses on analyzing employee attrition and predicting employees who are at risk of leaving an organization. Using Exploratory Data Analysis (EDA), Machine Learning, and Power BI, the project identifies key factors contributing to employee turnover and provides actionable insights for HR decision-making.

## Objectives

* Analyze employee attrition patterns.
* Identify factors influencing employee turnover.
* Build a machine learning model to predict attrition.
* Categorize employees into risk groups.
* Create an interactive Power BI dashboard for visualization and analysis.

## Tools & Technologies

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Power BI

## Dataset

IBM HR Analytics Employee Attrition Dataset

## Exploratory Data Analysis

The following analyses were performed:

* Attrition Distribution
* Department-wise Attrition Analysis
* Job Role-wise Attrition Analysis
* Monthly Income vs Attrition
* Overtime Analysis
* Correlation Heatmap

## Machine Learning Model

Model Used:

* Logistic Regression

Evaluation Metrics:

* Accuracy Score
* Classification Report
* Confusion Matrix

## Risk Analysis

Employees were classified into:

* Low Risk
* Medium Risk
* High Risk

based on predicted attrition probabilities generated by the machine learning model.

## Power BI Dashboard

The dashboard includes:

* Total Employees
* Employees Left
* Average Monthly Income
* Attrition by Department
* Attrition by Job Role
* Attrition by Overtime
* Employee Risk Distribution
* Interactive Slicers

## Key Insights

1. Employees working overtime showed higher attrition rates.
2. Certain job roles experienced significantly higher employee turnover.
3. Monthly income was an important factor influencing employee retention.
4. The predictive model successfully identified employees at risk of leaving.
5. Risk segmentation can help HR teams prioritize retention efforts.
